# W-Rapper 🎤

Turn your grocery list into a rap song. That's it.

## What it does

You type in whatever's on your grocery list — eggs, oat milk, whatever — pick a rap style, and it writes you a full 12–16 bar rap using all your items. Styles include Old School, Trap, Boom Bap, Drill, Conscious, and Comedy.

Built with plain HTML, CSS, and JavaScript. No frameworks, no build step, just three files.

## How to run it

1. Clone the repo
2. Open `app.js` and put your API key on line 8
3. Open `index.html` in a browser

That's literally it. No `npm install`, no terminal needed.

## API

Uses the [HackClub AI proxy](https://ai.hackclub.com) with the `qwen/qwen3-32b` model.
